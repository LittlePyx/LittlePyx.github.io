物联网技术与应用

Created2024-01-17|Updated2024-01-17
|Post View:

物联网的四个主要特征（P2-3）

  1. 全面感知
  2. 可靠传递
  3. 智能处理
  4. 深度应用

物联网得益于什么才得到巨大发展（P4，P9）

  • 随着芯片、传感器等硬件价格的不断下降，通信网络、云计算和智能处理技术的革新和进步，物联网迎来了快速发展期。

  • 物联网正是得益于云计算、大数据和人工智能的支持，才得到蓬勃发展，并为用户提供了更好的服务体验

物联网通过各种感知设备（如RFID、传感器、二维码等）感知物理世界的信息，这些信息通过互联网传输到云端存储设备中，为后续分析和利用提供支撑。

物联网感知的数据具有异构、多源和时间序列等特征，海量的感知数据具有典型的大数据特点，需要采用大数据分析技术进行深度分析和挖掘，为用户提供高效的数据应用服务

物联网感知的海量数据，包含人、机、物共融信息，对这些信息的深度挖掘、分析、利用、控制和可视化，离不开人工智能技术

物联网的安全技术包含哪些（第七章）

物联网的感知层安全：

  • RFID的安全机制（改变关联性方法、改变唯一性方法、隐藏信息方法、无线隔离方法、同步方法）
  • 物联网摄像头的安全机制（物联网摄像头风险分析、物联网摄像头安全措施）
  • 二维码的安全机制
  • 感知层的可信接入机制（信任管理、身份认证）

物联网的传输层安全：

  • 传输层的数据加密机制
  • 传输层的安全传输协议（IPSec协议、SSL安全协议、HTTPS安全协议）

物联网应用层安全

  • 访问控制（在身份认证的基础上，依据授权对提出的资源访问请求加以控制）
  • 数字签名
  • 外包数据的隐私保护（支持密文模糊检索的加密算法、同态加密）

物联网位置隐私

  • 位置匿名和混淆技术（位置k-匿名、假位置、空间数据加密）

条形码、二维码技术特点（P87,P93）

  • 条码技术拥有速度快、准确率高、可靠、寿命长、成本低廉等特点

条码：条码是由一组规则排列的条、空以及对应的字符组成的标记。“条”指对光线反射率较低的部分；“空”指对光线反射率较高的部分。这些条和空组成的数据表达一定的信息，并能够用特定的设备识读，以转换成与计算机兼容的二进制和十进制信息。

一个完整的条形码符号是由两侧空白区、起始字符、数据字符、校验字符、和中止字符组成。

  • 二维码技术：信息容量大、编码范围广、纠错能力强、可引入加密机制、易打印 寿命长 成本低、识读方便

二维条码的的长度、宽度均记载着数据。二维条码有一维条码没有的“定位点”和“容错机制”，容错机制在即使没有识别到全部的条码，或是条码有污损时，也能正确地还原条码上的信息

二维码技术与条码技术的一些共性：每种码型有其特定的字符集，每个字符占有一定的宽度，具有一定的校验功能，同时还具有对不同行的信息自动识别及处理旋转图形的功能。

北斗导航三号系统组成有哪些（P120）

由35颗卫星组成，包括5颗静止轨道卫星、27颗中地球轨道卫星、3颗倾斜同步轨道卫星，在空中连续发送带有时间和位置信息的无线电信号，供接收机接收

网络安全的根本目的（P215）

网络安全的根本目的就是防止网络传输过程中的信息被篡改、监听和非法使用

数据挖掘的常用手段（P192）

数据挖掘的常用手段大体可以分为三种：关联分析、分类分析以及聚类分析

关联分析：找出数据库中隐藏的关联网

分类分析：把数据库中的类标签未知的数据进行归类

聚类分析：将物理或抽象对象的集合分成由类似对象组成的多个类

智能电网是什么样的网络（P11）

智能电网以物联网为基础，其核心是构建具备智能判断和自适应调节能力的多种能源统一入网和分布式管理的智能化网络系统。通过对电网与用户用电信息进行实时监控和采集，采用最经济、最安全的输配电方式将电能输送到终端用户，实现对电能的最优配置与利用，提高电网运行的可靠性和能源的利用效率。从智能电网的能源接入、输配电调度、安全监控与继电保护、用户用电信息采集、计量计费到用户用电，都是通过物联网技术实现的。

近距离无线通信的特点是什么？（ChatGPT搜的，书上貌似没有总结，从P127开始）

1、灵活性：无线通信设备的位置不受固定网络设施的限制，可以在任何有无线信号覆盖的区域使用；

2、移动性：用户可以自由移动，同时保持与网络的连接；

3、安装便捷：无需进行大量的布线工作，只需安装一个或多个接入点设备，就可以建立覆盖整个区域的局域网络；

4、易于扩展：可以根据需要轻松地从只有几个用户的小型网络扩展到拥有上千用户的大型网络；

5、低成本：无线通信技术通常比有线通信技术成本更低，尤其是在需要大量布线或频繁调整网络布局的情况下；

6、易于维护：由于没有大量的有线线路，所以维护起来相对简单；

7、多通道通信：可以实现多通道同时通信，提高了通信效率；

8、高数据传输速率：可以实现高速数据传输，满足各种应用需求；

9、安全性高：可以采用加密技术等安全措施来保护数据的安全。

数字签名的概念以及它要解决的问题（P243）

  • 数字签名是指用户用自己的私钥对原始数据的哈希摘要进行加密所得的数据。

  • 信息接收者使用信息发送者的公钥对附在原始信息后的数字签名进行解密后获得哈希摘要，并通过与自己收到的原始信息的哈希摘要对照，可确信原始信息是否被篡改。

  • 数字签名技术是保证信息传输的保密性、数据交换的完整性、发送信息的不可否认性、交易者身份的确定性的一种有效的解决方案，是保障计算机信息安全性的重要技术之一。

数据预处理要处理哪些问题？这些数据的特点是什么？（P189-190）

  • 数据预处理要处理的问题有：

  1. 删去数据中重复的记录、消除数据中的噪声数据、纠正不完整和不一致的数据；

  2. 将来自多个数据源的数据合并到一起构成一个完整的数据集；

  3. 将一种格式的数据转换为另一种格式的数据；

  4. 在尽可能保持数据原貌的前提下，最大限度地精简数据量。

  • 这些数据的特点是：

1、含有噪声；2、不完全的；3、不一致的。

物联网、云计算、大数据、人工智能的相互关系是什么？（P9）

物联网、云计算、大数据和人工智能是一脉相承的。其中，物联网是数据获取的基础，云计算是数据存储的核心，大数据是数据分析的利器，人工智能是反馈控制的关键。物联网、云计算、大数据和人工智能构成了一个完整的闭环控制系统，将物理世界和信息世界有机融合在一起。

蓝牙、ZigBee技术的组网特点（网上的答案，对P130-136的总结）

  • 蓝牙技术是一种低成本、短距离的无线个人网络传输技术，其主要目标是提供一个全世界通行的无线传输环境，以通过无线电波来实现所有移动设备之间的信息传输服务。

  • 蓝牙模块组网技术的主要特点如下：

  1. 支持移动联网。用户可以像使用移动电话那样灵活的移动设备的位置，而使设备仍然保持持续的网络连接；

  2. 不需要使用物理线路，安装非常简便，成本低、体积小，可用于更多的设备；

  3. 网络使用的高频电波可以穿透墙壁或玻璃窗，所以设备可以在有效范围任意放置；

  4. 多层安全防护措施可以确保用户隐私；

  5. 改动网络结构或布局时，不需要对网络进行重新设置。

  • ZigBee是IEEE 802.15.4协议的代名词。根据这个协议规定的技术是一种短距离、低功耗的无线通信技术。

  • 主要特点如下：1、自动组网，网络容量大；2、网络时延短；3、模块功耗低，通讯速率低；4、传输范围小；5、成本低；6、可靠性好，安全性高。

物联网大数据的特征有哪些？（P176）

1、海量性；2、异构性和多态性；3、实时性和动态性；4、关联性及语义性；5、准确性和真实性。

卫星系统的分类方法有哪些？（P140-141）

  1. 按卫星制式可分为静止卫星通信系统、随机轨道卫星通信系统和低轨道卫星通信系统；

  2. 按通信覆盖区域的范围可划分为国际卫星通信系统、国内卫星通信系统和区域卫星通信系统；

  3. 按用户性质可分为公用（商用）卫星通信系统、专用卫星通信系统和军用卫星通信系统；

  4. 按业务范围可分为固定业务卫星通信系统、移动业务卫星通信系统、广播业务卫星通信系统和科学实验卫星通信系统；

  5. 按基带信号体制可分为模拟制卫星通信系统和数字制卫星通信系统；

  6. 按多址方式可分为频分多址、时分多址、空分多址和码分多址卫星通信系统；

  7. 按运行方式可分为同步卫星通信系统和非同步卫星通信系统。目前国际和国内的卫星通信大都是同步卫星通信系统。

简述 EPC 系统的组成（P29）

EPC 系统由 EPC 编码体系、射频识别系统和信息网络系统三部分组成。

光电传感器的概念（P72）

光电式传感器就是将光信号转化为电信号的一种器件，简称为光电器件。

常用的定位方法（P115）

  • 卫星定位技术

  • 蜂窝定位技术（COO定位、TOA定位、TODA定位、AOA定位、A-GPS定位）

  • wifi定位技术（质心定位算法、位置指纹识别算法）

常用的近距离无线通信技术（P127）

近距离无线通信技术有蓝牙(Bluetooth)、无线局域网802.11(Wi-Fi)、ZigBee。

物联网面临的安全问题有哪些（P211）

物联网作为一个人机物融合系统，面临的安全问题十分复杂，主要包括：物联网感知层安全问题（接入安全、物理安全、系统安全、通信安全）、物联网传输层安全问题、物联网应用层安全问题等。

智能家居、三网融合的概念（P257）

  • 基于物联网的智能家居，表现为利用信息传感设备（同居住环境中的各种物品松耦合或者紧耦合）将与家居生活有关的各种子系统有机地结合在一起，并与互联网连接起来，进行监控、管理信息交换和通信，实现家居智能化，包括：智能家居（中央）控制管理系统、终端（家居传感器终端、控制器）、家庭网络、外联网络、信息中心等。
  • 三网融合是指电信网、广播电视网和互联网的融合，即将传统的电信、广播电视和互联网三个独立运营的网络进行整合和融合，实现资源共享、业务互通、技术融合的发展模式。通过三网融合，可以提高网络资源的利用效率，推动信息化建设的深入发展。（百度解释）

视频检索、蓝牙的概念（P208、P130）

  • 视频检索指根据用户提出的检索请求，从视频数据库中快速地提取出相关的图像或图像序列地过程。
  • 蓝牙（Bluetooth）是一种支持设备短距离通信（10cm-10m)的无线电技术，能在包括移动电话、PDA、无线耳机、笔记本电脑、相关外设等众多设备之间进行无线信息交换。

传感器、北斗导航的概念（P41 P120）

  • 在为数众多的敏感元件中，我们把那些能将非电量形式的参量转换为电参量的元器件叫作传感器。
  • 北斗卫星导航系统是我国自主研发、独立运行的卫星导航系统。

物联网在工业流程管理中的作用和意义（P269、271）

  • 作用：工业是物联网应用的重要领域。具有环境感知能力的各类终端、基于泛在技术的计算模式、移动通信等不断融入工业生产的各个环节，可大幅提高制造效率，改善产品质量，降低产品成本和资源消耗，将传统工业提升到智能工业新阶段。

  • 意义：物联网是信息通信技术发展的新一轮制高点，正在工业领域获得广泛的应用，并与未来先进制造技术相结合，形成新的智能化的制造体系。物联网在工业领域的应用主要集中在以下几个方面：制造业供应链管理、生产过程工艺优化、产品设备监控管理、环保监测及能源管理、工业安全生产管理。

物联网安全问题和特征是什么（P211-213、2）

  • 安全问题：物联网感知层安全问题（接入安全、物理安全、系统安全、通信安全），物联网传输层安全问题，物联网应用层安全问题。
  • 特征：全面感知、可靠传递、智能处理、深度应用

基于物联网的智能物流管理关键技术有哪些（P267、268）

  1. 智能物流管理系统在物理层上采用RFID技术；

  2. 智能物流管理系统在软件体系结构可分为四个层面：第一层是云计算平台；第二层是基于人工智能方法的云计算应用程序，可称之为智能引擎；第三层是业务逻辑层；第四层是直接为用户或管理员提供可视化的操作、查询的Web界面。

  3. 云计算是物联网整个智能物流管理系统体系结构的中枢与连接的纽带。

  4. 以EPC码和物联网为核心在互联网之上构造“物联网”。

  5. 关键技术涉及以下几个方面：实现信息采集、信息处理的自动化；实现商品实物运动等操作环节的自动化；实现管理和决策的自动化乃至智能化

  1. 数据的海量性

  2. 数据的异构性和多态性

  3. 数据的实时性和动态性

  4. 数据的关联性及语义性

  5. 数据的准确性和真实性

什么是无线传感网络，无线传感网络的组成是什么（P27）

  • 无线传感器网络（WSN）是由大量的密集部署在监控区域的智能传感器节点构成的一种网络应用系统。

  • 无线传感器网络的组成：由大量功能相同或不同的无线传感器节点、接收发送器、Internet或通信卫星、任务管理节点等部分组成的一个多跳的无线网络。
